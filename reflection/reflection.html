Week-1

Online course di Minggu pertama diperkenalkan aplikasi Slack group, ini pertama kalinya menggunakan aplikasi group seperti ini. Lanjut baca materi dan kerjakan tugas tentang pengenalan Git dan Github HTTPS dan SSH yang merupakan pengalaman baru. Mencoba mengenal command nya, berinteraksi dan coba upload. Sempat salah membuat folder dan salah upload. Baca materinya lagi, variatif (membuat bingung pilih yang mana) dan ikuti serta googling informasi terkait lainnya, coba lagi dan lagi sampai bisa. Senang dapat ilmu baru yaitu mempelajari konsep Git/Github, HTML5 beserta elemen nya, CSS3 dengan transisi dan transformasinya, analisa wireframing. Saat coding awal saya memikirkan bagaimana agar lebih cepat memahami materi agar mudah untuk implementaskan tugas demi tugas harian untuk dikerjakan, hal yang perlu dihindari adalah kurangnya waktu buat keluarga (Istri dan Anak-anak) karena sibuk baca materi dan praktek ngoding, sampai harus ngoding di waktu subuh (moga ga telat lagi submit tugas harian). Hardwork and sacrifice maybe. Lesson to learn that work and life should be balance....Learn to code and life should be balance. Mentor yang remind dan memberikan response dan suggest, almost feel like pair programming. It'll will be nice to have Team work moment to achieve on something.

"Work and life should be balance"
Still a long way to run....Hopefully, in the next ahead would be anticipate, learn faster, achive next challenges, get skilled and ready to implement. Maju terus BlandfordFox...

Week-2

Pada minggu kedua ini memasuki pembelajaran tentang Javascript, Javascript adalah bahasa pemrograman yang cukup populer saat ini. Pelatihan untuk para peserta adalah dengan mengakses ke web tutorial CodeAcademy Melalui web ini peserta dihadapkan dengan pembelajaran secara praktek yang terdiri dari 8 tahap. Tahap demi tahap dilalui dengan pelbagai tingkat kesulitan masing-masing. Website ini juga menyediakan link forum sebagai media bertanya & menjawab pertanyaan tugas-tugas praktek tiap-tiap tahap. Dalam tahap pembelajaran ini mengajarkan kepada kita bahwa pembelajaran coding adalah sejauh praktek dari coding itu sendiri.

Semakin lama & panjang praktek coding yang dilakukan akan sangat membantu meningkatkan pengalaman & pengetahuan coding.

Lagi-lagi improvement dalam time management diperlukan untuk melaluinya.

Pengalaman pembelajaran Week ke-2 ini dapat digunakan untuk menunjang kelanjutan dari pembalajaran di minggu2 selanjutnya. Tetap semangat dan keep learning dimana diperlukan pengalaman serta pengetahuan yang dibutuhkan seseorang untuk dikatakan sebagai seorang yang ahli coding.
