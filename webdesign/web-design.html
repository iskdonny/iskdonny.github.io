Get to Know Adaptive Web Design (AWD) dan Responsive Web Design (RWB)

iskdonny.github.io

Apa itu adaptive dan responsive web design?

Adaptive design adalah design layout web yang sudah ditentukan untuk memberikan tampilan pada sebuah website. Developer saat coding perlu mengetahui website ini digunakan untuk diakses dengan device lebar screen berapa saja. Responsive design adalah design web yang memberikan perubahan ukuran web yang smooth pada browser device.

Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

Responsive design bagus mengingat tampilan website lebih smooth serta compatible dengan screen device yang beredar saat ini serta pertimbangan waktu yang lebih singkat yang dibutuhkan developer untuk coding karena tidak harus men define secara specific untuk web layoutnya untuk diakses device tertentu.

Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

Hadirnya Smartphone dan tablet mempengaruhi design web yang ada. Technology baru akan berpengaruh dengan design web yang ada seperti augmented reality dsb

Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

Akan ada keterbatasan untuk berinteraksi dengan kebanyakan website. selama kekurangan fisiknya mampu menggerakan anggota tubuh untuk berinteraksi dengan website. Technology hardware seperti sensor dan suara akan sangat membantu para disabilitas untuk berinteraksi dengan website/internet.

Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web?

Saat ini konsep yang sesuai adalah responsive web namun semoga kedepannya ada solusi responsive web yang hemat loading serta ruang penyimpanan file terutama image/video.
