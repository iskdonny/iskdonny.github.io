Recap of Week 1

Posted on September 04, 2016 at 13:00 AM

Awal-awal adalah berkenalan menggunakan Slack sebagai media diskusi. Aplikasi yang bagus juga canggih untuk saling chat dan share image. Selanjutnya mengenal Github sebagai media repository dan mengaksesnya langsung melalui command based (Git). Mempersiapkan folder untuk mengupload tugas dan mengenal file markdown. Git ini sangat berguna untuk mengupload static website yang merupakan tugas harian.

Lalu HTML (Hypertext markup language) yang digunakan saat ini adalah versi 5. HTML dipergunakan untuk membuat struktur halaman website yang nantinya diakses menggunakan browser melalui HTTP protokol. HTML diupload di folder github repository sebagai free hosting website. Lalu ditampilkan di website sesuai dengan nama yang diinginkan diikuti dengan default domain github.io.Click here


CSS3 membuat tampilan serta layout web tampil lebih baik, direkomendasikan penggunaan CSS dalam HTML dimana kedua adalah file terpisah. Dengan CSS external maka kode HTML dapat diminimalkan jumlah barisnya. Serta CSS dapat digunakan untuk membuat efek-efek seperti animasi di website kita


Selanjutnya membuat HTML form dan first time blog website. Tantangan di sini adalah memadukan elemen HTML dan fitur dari CSS. Sehingga dapat menampilkan blog site yang baik strukturnya, menarik layoutnya juga belajar menulis content dalam blog itu sendiri.


Kegiatan pada hari terakhir dalam week-1 adalah meninjau proses pembuatan website, mengenal Adaptive serta Responsive web design, evaluasi tampilan dari sebuat website dan wireframing dan yang terakhir adalah explore core values. Banyak hal teknis dan non teknis yang perlu dilakukan improvement dari seminggu yang dipelajari dari bootcamp ini.
